========================================
    畜牧業管理系統 - Windows快速部署
========================================

🚀 快速啟動（推薦）
=====================================

1. 運行: quick-start.bat
   - 自動檢查環境
   - 自動安裝依賴
   - 直接啟動系統

📦 FFmpeg安裝（影像串流功能）
=====================================

方式一：使用Node.js安裝（推薦）
npm run install-ffmpeg

方式二：使用批次腳本
setup-ffmpeg.bat

方式三：手動安裝
install-ffmpeg-only.bat

🎯 系統地址
=====================================

主控台: http://localhost:3000
攝影機管理: http://localhost:3000/onvif-cameras

🔧 可用腳本
=====================================

quick-start.bat          - 快速啟動（推薦）
setup-ffmpeg.bat         - 安裝FFmpeg
setup-cameras.bat        - 批量連接攝影機
test-cameras.bat          - 測試攝影機連接

📋 npm命令
=====================================

npm start                - 啟動系統
npm run install-ffmpeg   - 安裝FFmpeg
npm run setup            - 完整安裝（依賴+FFmpeg）

🛠️ 故障排除
=====================================

問題：腳本閃退
解決：使用 quick-start.bat 或 deploy-windows-debug.bat

問題：攝影機無法連接
解決：檢查IP和認證資訊，運行 test-cameras.bat

問題：串流無法播放
解決：確保FFmpeg已安裝，運行 npm run install-ffmpeg

🎉 完成！
=====================================

系統部署完成後，您可以：
- 管理32台攝影機
- 查看即時影像
- 拍攝快照
- 批量操作
